题目分析

进行推导:

$\frac{1}{x} + \frac{1}{y} = \frac{1}{n!}$

$\Rightarrow \frac{x + y}{xy} = \frac{1}{n!}$

$\Rightarrow (x + y)n! = xy$

$\Rightarrow xy - (x + y)n! = 0$

$\Rightarrow (n!)^{2} - (x + y)n! + xy= (n!)^{2}$

$\Rightarrow (x - n!)(y - n!) = (n!)^{2}$

令$a = (x - n!)$,$b = (y - n!)$.

$ab = (n!)^{2}$

由算术基本定理知:

$n! = p^{c{1}}{1}p^{c{2}}{2} \cdots p^{c{n}}{n}$

$\Rightarrow (n!)^{2} = p^{2c{1}}{1}p^{2c{2}}{2} \cdots p^{2c{n}}{n}$

因为$a = (x - n!)$,其中$n!$是固定的,所以求出了$a$也就求出了$x$.

又因为$ab = (n!)^{2}$,得出$a$是$(n!)^{2}$的一个因数.

由算术基本定理知,$(n!)^{2}$的正因数的个数为:

$\sigma{1}((n!)^{2}) = (1 + 2c{1})(1 + 2c{2}) \cdots (1 + 2c{n})$

再根据阶乘分解的思想,$(n!)^{2}$中质因子$p$的个数等于$1 \to n$中每一个数中包含$p$的个数,显然,在$1 \to n$中,包含一个$p$的有$\lfloor \frac{n}{p} \rfloor$个,包含两个$p$的有$\lfloor \frac{n}{p^{2}} \rfloor$个,不过因为其中一个已经在$\lfloor \frac{n}{p} \rfloor$里计算过,所以累加上$\lfloor \frac{n}{p^{2}} \rfloor$即可,而不是加上$2 \times \lfloor \frac{n}{p^{2}} \rfloor$.

所以$(n!)^{2}$中$p$的个数为$\sum{p{k}^{} \leq n} \lfloor \frac{n}{p^{k}} \rfloor$.

综上,用一个线筛筛出$n$以内的所有素数,算出每个素数的$c$,再按照上式计算即可科技为了我.
